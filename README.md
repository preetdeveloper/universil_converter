## 📌 About

This is a simple Universal File Converter built with Python.
It allows basic file conversions like:

* JPG → PNG
* PNG → JPG
* and other simple formats

This project helps beginners understand:

* File handling
* Image processing
* GUI design

---

## 🚀 Features

* Easy file selection
* One-click conversion
* Simple and clean UI
* Beginner-friendly code

---

## 🛠️ Requirements

```bash
pip install pillow
```

## ▶️ How to Run (Python)

1. Download or clone this project
2. Open terminal in project folder
3. Run:

